# Plugin Inteligente de Análise Evolutiva de Código para o Visual Studio Code

## Objetivo
Registrar a evolução do código ao longo do tempo e analisar o comportamento do programador, com o objetivo de fornecer sugestões personalizadas para melhoria da qualidade do código e das práticas de desenvolvimento.

## Público-Alvo 
Estudantes de programação e desenvolvedores junior que buscam “mentoria automatizada”, com feedback contínuo durante o processo de desenvolvimento.

## Tecnologias
JavaScript
